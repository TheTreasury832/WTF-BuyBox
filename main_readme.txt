- Analyze multifamily, SubTo, seller finance deals

- Generate investor-ready summaries

### 👑 Empire ScriptMaster AI

## 📦 Quick Start

### Prerequisites
- Python 3.8+
- pip or conda
- Git

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/yourusername/wholesale2flip-cartel.git
cd wholesale2flip-cartel
```

2. **Choose your version:**

**For V1 FREE:**
```bash
cd v1-free
pip install -r requirements.txt
streamlit run app_v1.py
```

**For PRO:**
```bash
cd pro
pip install -r requirements.txt
streamlit run app_pro.py
```

3. **Open your browser:**
- Local: `http://localhost:8501`

## 🚀 Deployment

### Streamlit Cloud (Recommended)

1. **Fork this repository**
2. **Connect to Streamlit Cloud:**
   - Go to [share.streamlit.io](https://share.streamlit.io)
   - Connect your GitHub account
   - Select your forked repository

3. **Deploy V1 FREE:**
   - Main file: `v1-free/app_v1.py`
   - Advanced settings: Add `v1-free` as the main folder

4. **Deploy PRO:**
   - Main file: `pro/app_pro.py`
   - Advanced settings: Add `pro` as the main folder

### Alternative Deployments
- **Heroku:** See [deployment guide](./docs/deployment.md)
- **AWS/GCP:** Docker configurations available
- **Local:** Use provided scripts in `/scripts`
